# car_drivers API
 Dart Aqueduct Server
 
 БД PostgreSQL,
 Авторизация OAuth 2
 
 Контроллеры в lib,
 конфигурация в bin/main.dart
 
 1) Контроллер для водителей CRUD (без U)
 2) Контроллер для тачек CRUD (без U)
 3) Контроллер с одним GET для получения рандомной пары водитель + тачка
